# Life Expectancy Analysis

## Project Overview
Analysis of global life expectancy trends from 1800 to present.

## Dataset Description

The dataset contains historical and projected **life expectancy data** for countries around the world, spanning the years **1800 to 2100**. Each record represents the average life expectancy for a specific country in a given year, enabling the analysis of long-term health trends across different regions.

### Dataset Features

The dataset includes the following key variables:

| Feature | Description |
|---------|-------------|
| `name` | Country name |
| `year` | The observation year (1800–2100) |
| `life_expectancy` | Average life expectancy at birth, measured in years |

### Dataset Insights

This dataset provides valuable insights into how improvements in:

- Healthcare
- Nutrition
- Sanitation
- Education
- Socioeconomic development

have contributed to the increase in life expectancy worldwide over time.

By analyzing historical trends and future projections, we can explore differences between countries and regions and understand global health progress.


## Questions Explored
1. How has life expectancy changed globally?
2. Which continents have improved the most?
3. What factors influence life expectancy?
4. What is the future projection?
